# 二、100个维度自我分析

## 2.1 视觉设计 (1-15)

### 1. 色彩系统
- 主色：蓝色 #2563eb
- 辅助色：渐变紫 #7c3aed
- 背景色：#f9fafb
- 文字色：#374151 / #6b7280 / #9ca3af
- 状态色：成功#10b981 / 警告#f59e0b / 错误#ef4444
- **问题**：蓝色太常见，缺乏品牌辨识度
- **改进**：引入品牌主色（如渐变蓝紫+橙色点缀）

### 2. 字体系统
- 标题：system-ui / -apple-system
- 正文：system-ui
- 代码：ui-monospace
- **问题**：字体没有层次感
- **改进**：引入字体大小规范

### 3. 间距系统
- 基准单位：4px
- 间距梯度：4/8/12/16/20/24/32/48
- **问题**：间距不统一
- **改进**：建立8px间距体系

### 4. 圆角系统
- 按钮：8px
- 卡片：12px
- 对话框：16px
- 头像：50%
- **问题**：圆角不统一
- **改进**：统一8px圆角规范

### 5. 阴影系统

## 4.2 组件库

### 基础组件
- Button (primary/secondary/ghost/danger/text)
- Input (text/password/textarea/select/checkbox/radio/switch)
- Card (bordered/elevated/filled)
- Badge (status/count/dot)
- Avatar (image/icon/string/group)
- Tooltip (top/bottom/left/right)
- Progress (linear/circular)
- Skeleton (text/avatar/card/table)
- Empty (illustration/action)
- Spinner (sm/md/lg)
- Divider (horizontal/vertical)

### 复合组件
- Modal (confirm/dialog/drawer)
- Dropdown (menu/context)
- Select (single/multi/async)
- DatePicker (date/time/range)
- Cascader (联动选择)
- AutoComplete (智能补全)
- Transfer (穿梭框)
- ColorPicker (颜色选择)
- Upload (单文件/多文件/拖拽)
- Form (动态表单)

### 业务组件
- JDMatcher (首页)
- ResumeEditor (编辑器)
- ResumePreview (预览)
- TemplateSelector (模板选择)
- AIGenerateButton (AI优化)
- HistoryButtons (历史版本)
- ExportButton (导出)
- VersionSelector (版本选择)
